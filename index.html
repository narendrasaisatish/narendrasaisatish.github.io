Deep Noise Suppression using Autoencoders

This project showcases the application of autoencoders for image denoising. The model was trained on the MNIST dataset to remove various types of noise, including Gaussian, salt-and-pepper, and speckle noise, restoring corrupted images effectively.

  • Libraries Used: PyTorch, Torchvision, NumPy, Matplotlib
  • Concepts Involved: Autoencoders, Image Denoising, Convolutional Neural Networks (CNNs)
  • Results: The autoencoder successfully denoised images, handling different noise types, with visualizations showing improved image quality.
